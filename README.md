# Axis-Template
Focused on building a scalable layout, clean CSS structure, and a fully responsive experience across devices

### Features ✨
<ul>
  <li>Semantic HTML5 Structure</li>
  <li>Responsive Layout (Mobile First)</li>
  <li>CSS Grid & Flexbox</li>
  <li>Fluid Typography using clamp()</li>
  <li>Clean UI with smooth hover effects</li>
</ul>


🚀 Purpose

This project was built to strengthen my core front-end fundamentals and practice writing scalable, organized CSS without relying on frameworks.
